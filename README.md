# 🔐 Password Strength Checker

## 🇬🇧 English

Password Strength Checker is a beginner-friendly C++ console application that evaluates the security of a user's password.

The program analyzes password length, uppercase letters, lowercase letters, numbers, and special characters to determine password strength. It also provides suggestions to help users create stronger and more secure passwords.

This project was created to practice fundamental C++ programming concepts and basic cybersecurity principles.

### 🎮 Features

* Password strength analysis
* 5 security levels:

  * Very Weak
  * Weak
  * Medium
  * Strong
  * Very Strong
* Checks password length
* Detects uppercase and lowercase letters
* Detects numbers and special characters
* Provides security improvement suggestions
* Beginner-friendly C++ structure

### 🕹️ How to Use

1. Run the program
2. Enter a password
3. View the password strength level
4. Read the suggestions for improving password security

### 🚀 Future Improvements

* Password entropy calculation
* Dictionary word detection
* Common password blacklist
* Password generator feature
* GUI version using Raylib or SFML
